TestUtil uses exceptions for Warnings, Glitches, Cautions, Skips and "fatal" exceptions. The abstract exceptions are:
  • TestUtilRuntimeException (TestUtil base class)
  • TestUtilCautionException
  • TestUtilFatalException
  • TestUtilGlitchException
  • TestUtilSkipException (not abstract)
  • estUtilWarningException
These exceptions are inherited as required.
